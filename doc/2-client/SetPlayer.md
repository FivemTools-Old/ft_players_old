# SetPlayer

SetPlayer(attribut, value) -- Set specific attribut
SetPlayer({ {attribut, value} , {...} }) -- Set specific attribut

## Description

Function to set a value to an attribute of a player.

## Exemples

```lua
SetPlayer("steamId", "steam:1100001xxxxxxxx") -- Set specific attribut
SetPlayer({ {"steamId", "steam:1100001xxxxxxxx"} }) -- Set specific attribut
```
